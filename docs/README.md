## どうやって接続するの？
> [!TIP]
> アクセス用のipについてはDiscordサーバーをご覧ください。  
> **Q.** ipとport(ポート)ってなんぞや？どうやって参加するんですか？ip入力しても入れません！  
> **A.** ipの後ろにportを記述します。**{:}に注意！**  
> ```
> 例）ip: 'Thisisatestip'
>   port: '12345'
> 'Thisisatestip:12345' に接続する
> ```
